# simulador_de_financas
Arquivo criado como atividade do curso de Excel pela DIO.

Descrição
Esta planilha foi criada para treinamento em habilidades com Excel, simulando operações bancárias e controle pessoal de receitas. O objetivo é proporcionar uma experiência prática, permitindo ao usuário desenvolver competências em organização financeira, automação de tarefas e navegação eficiente.
Funcionalidades

Simulação de operações bancárias: Controle de receitas, despesas, saldo e movimentações financeiras.
Gestão pessoal de receitas: Registro e acompanhamento de entradas e saídas, categorização de gastos e análise de fluxo de caixa.
Menu de navegação otimizado: Interface intuitiva para facilitar o acesso às principais funcionalidades e abas da planilha.
Automação com macros: Automatização de tarefas repetitivas, tornando o uso mais eficiente (necessário habilitar macros).
Exercícios práticos: Ideal para treinar fórmulas, funções, gráficos e recursos avançados do Excel.

Estrutura da Planilha

Abas principais:

Controle de receitas e despesas
Relatórios financeiros
Dashboard com gráficos
Menu de navegação


Macros:

Automatização de lançamentos
Geração de relatórios
Navegação entre abas



Como usar

Abra o arquivo PRojeto2_DIO.xlsm no Excel.
Habilite as macros para acessar todas as funcionalidades automatizadas.
Utilize o menu de navegação para acessar as diferentes áreas da planilha.
Registre suas operações financeiras conforme o modelo proposto.
Explore as funcionalidades para treinar fórmulas, gráficos e automações.
Salve seu progresso e utilize a planilha como base para aprimorar suas habilidades.

Requisitos

Microsoft Excel (preferencialmente versão 2016 ou superior)
Habilitação de macros (VBA)
